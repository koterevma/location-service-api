/insert
POST with header "Content-Type: application/json"

insert values to database
format for datetime: "YYYY-MM-DD HH:MM:SS" (space can be swapped by 'T', see examples below)
format for coords: "1.12,2.21" (precision up to 8 digits after a point)

/get[?from=datetime][&to=datetime]
returns a list of all values from date1 to date2 in json format

/get/last[?count=N]
returns N last records. If count isn't specified returns last record


Examples


localhost for example urls should be replaced with server ip/domain on which api is hosted
http://localhost/insert POST '{"latitude":12, "longitude":11, "date": "2021-10-17T15:44:00"}'
http://localhost/get
http://localhost/get?from=2021-10-17T15:44:00
http://localhost/get?from=2021-10-17T15:44:00&to=2021-10-17T15:47:00
http://localhost/get/last
http://localhost/get/last?count=4
